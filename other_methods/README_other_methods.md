# other_methods — reference cases for non-EVC methods

Standalone demos of methods this library teaches *about* but does not promote:
they're here for completeness and honest comparison, not prominence.

| File | Method |
|---|---|
| [`_main/RCV_ballot_example.yaml`](_main/RCV_ballot_example.yaml) | RCV-IRV — a plain ranked-ballot instant-runoff count |
| [`_main/03a_stv_3seats.yaml`](_main/03a_stv_3seats.yaml) | STV — proportional seats from ranked ballots (compare [`../03_STAR_PR/`](../03_STAR_PR/) on score ballots) |

Most non-EVC material lives where it teaches best: inside
[`../method_comparisons/`](../method_comparisons/), where RCV-IRV and
choose-one serve as the foil to the equal-vote methods. Concept docs:
[`../00_start_here/RCV_IRV/`](../00_start_here/RCV_IRV/) and
[`../00_start_here/other_ranked_methods/`](../00_start_here/other_ranked_methods/).
